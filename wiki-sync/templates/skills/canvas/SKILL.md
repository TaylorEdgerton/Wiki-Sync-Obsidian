# canvas

Use this skill to create or update Obsidian canvas files under `{{SYNC_ROOT}}/wiki/canvases/`.

## Operations

- `new`: create a blank named canvas
- `add`: place notes, text cards, images, or PDFs
- `zone`: create labeled groups for related material
- `from-topic`: build a topic map from existing related notes
- `from-site`: build a customer or service overview canvas

## Expectations

- Keep the canvas filename aligned with the topic slug.
- Link the canvas from the relevant wiki note or site index.
- Log the creation in `{{SYNC_ROOT}}/activity/`.
