# save

Use this skill to preserve durable insights from conversations or sessions.

## Workflow

1. Decide whether the note belongs in `{{SYNC_ROOT}}/wiki/concepts/`, `{{SYNC_ROOT}}/wiki/entities/`, `{{SYNC_ROOT}}/wiki/comparisons/`, `{{SYNC_ROOT}}/wiki/questions/`, or another domain area.
2. Write a note with full frontmatter and clear related links.
3. Update `{{SYNC_ROOT}}/wiki/index.md` and `{{SYNC_ROOT}}/wiki/meta/active-topics.md` if the note changes current priorities.
4. Append a timestamped activity log entry in `{{SYNC_ROOT}}/activity/`.

## Guardrails

- Save concise, reusable knowledge rather than chat transcripts.
- Link out to `{{SYNC_ROOT}}/sources/manifest.md`, `{{SYNC_ROOT}}/runbooks/`, `{{SYNC_ROOT}}/projects/`, `{{SYNC_ROOT}}/sites/`, and the current profile work queue when relevant.
