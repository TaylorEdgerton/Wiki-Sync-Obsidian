# obsidian-markdown

Use this skill as the style and syntax reference for notes written into the wiki.

## Preferred Syntax

- wikilinks: `[[Page Name]]`
- aliases: `[[Page Name|Label]]`
- callouts: `> [!info]`, `> [!warning]`, `> [!tip]`, `> [!contradiction]`
- frontmatter: flat YAML with no nested status workflow fields
- embeds: `![[File Name]]` when helpful

## Writing Rules

- Keep prose declarative and scannable.
- Use headings for durable sections, not for visual spacing alone.
- Prefer related links and tags over duplicate content.
